Reading Documentation is Hard

But it doesn't have to be
Read OpenAPI docs in your web browser

What is this

Reading API docs shouldn't be hard. I'm making it easier by displaying docs in this web application.

How it works

Type or paste an OpenAPI doc URL into the input above and click "Open Doc". The API will load in this area of the browser. You can use it with any public API doc online, as long as it's formatted in JSON or YAML. This includes documentation produced in Swagger and Apiary.

You can even open docs hosted on GitHub or BitBucket.
